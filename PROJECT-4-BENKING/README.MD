Banking System in C++ using OOP 

Project Description

This project is a console-based Banking System developed using C++.
It is designed to demonstrate Object-Oriented Programming concepts such as inheritance, polymorphism, abstraction, and encapsulation through a real-world banking example.

Project Objective

The objective of this project is to understand and implement OOP concepts in C++ by creating different types of bank accounts and performing basic banking operations.

OOP Concepts Used

Class and Object

Inheritance

Polymorphism

Encapsulation

Virtual Functions

Abstract Class

Class Structure

Base Class:

BankAccount (Abstract Class)

Derived Classes:

SavingsAccount

CheckingAccount

FixedDepositAccount

Features of the System

Create Savings, Checking, and Fixed Deposit accounts

Deposit amount into an account

Withdraw amount from an account

Display account details

Calculate interest based on account type

Menu-driven console application

Technologies Used

Programming Language: C++
Concepts: Object-Oriented Programming, STL (vector), Dynamic Memory Allocation

How to Run the Program

Compile the program using any C++ compiler

Run the executable file

Follow the menu options displayed on the screen

Learning Outcomes

Clear understanding of abstract classes and virtual functions

Practical experience with runtime polymorphism

Better understanding of inheritance and encapsulation in C++
